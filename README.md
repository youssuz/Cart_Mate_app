# CartMate

Kotlin과 Room Database 및 RecyclerView를 활용하는 방법을 학습하기 위해 제작한 장바구니 상품 관리 Android 애플리케이션입니다.

사용자는 상품을 등록, 수정, 삭제할 수 있으며, 장바구니 안 상품의 총 금액을 확인할 수 있습니다.

## Features & Tech Strack

* 상품 등록 / 수정 / 삭제
* 상품 목록 조회 (RecyclerView, ListAdapter, Flow)
* 선택된 상품 총 금액 계산
* 로컬 데이터 저장(Room Database)
